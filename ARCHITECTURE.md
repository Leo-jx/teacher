# 系统架构文档

## 概述

程序员AI辅助助手是一个可选认证的全栈应用，使用Cloudflare Pages + Workers + D1构建，提供专业的编程技术问答服务。

## 核心原则

1. **可选认证**：用户无需登录即可使用核心功能
2. **安全第一**：所有认证流程使用行业最佳实践
3. **数据隔离**：未登录用户数据不持久化到数据库
4. **边缘优化**：充分利用Cloudflare的边缘计算能力

## 核心模块详解

### 1. 认证模块 (Authentication)

#### 文件位置
- 前端: `public/js/app.js` (JWT存储、UI交互)
- 后端: `public/_worker.js` (认证逻辑)

#### 安全机制
- **加盐密码哈希**:
  ```javascript
  salt = 随机16字节
  hash = SHA-256(salt + password)
  存储格式: salt:hash
  ```

- **JWT Token**:
  ```
  Header: { alg: "HS256", typ: "JWT" }
  Payload: { userId, username, exp: 7天后 }
  Signature: HMAC-SHA256(secret, header.payload)
  ```

#### 流程

**注册流程**:
```
用户填写表单 → 输入验证 → 生成盐值 → 哈希密码 
    → 存入数据库 → 生成JWT → 返回用户
```

**登录流程**:
```
用户输入凭证 → 数据库查询 → 验证哈希 → 更新登录时间
    → 生成JWT → 返回用户
```

### 2. 聊天模块 (Chat)

#### 两种模式

**未登录模式**:
- 数据仅存储在浏览器内存
- 刷新页面后数据丢失
- 不调用数据库操作

**已登录模式**:
- 消息同步到D1数据库
- 支持跨设备访问
- 完整的对话历史管理

### 3. 数据库设计

#### Users 表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| username | TEXT | 用户名，唯一 |
| email | TEXT | 邮箱，唯一 |
| password_hash | TEXT | 加盐哈希后的密码 |
| created_at | DATETIME | 创建时间 |
| last_login | DATETIME | 最后登录时间 |

#### Conversations 表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| user_id | INTEGER | 外键 → users.id |
| title | TEXT | 对话标题 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

#### Messages 表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| conversation_id | INTEGER | 外键 → conversations.id |
| role | TEXT | 角色 (user/assistant) |
| content | TEXT | 消息内容 |
| created_at | DATETIME | 创建时间 |

## 安全设计

### 1. 传输安全
- 强制 HTTPS (Cloudflare 自动处理)
- TLS 1.3 加密

### 2. 认证安全
- 密码加盐哈希
- JWT 7天过期
- 无状态认证，避免会话劫持

### 3. 输入安全
- 正则表达式验证
- 输入长度限制
- SQL注入防护 (参数化查询)
- XSS防护 (CSP头 + 前端清理)

### 4. 数据安全
- 未登录用户数据不持久化
- 用户间数据完全隔离
- 使用外键约束保证完整性
